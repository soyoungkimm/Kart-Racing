## 프로젝트 소개 :smile:

프로젝트명 : Kart Racing<br>
제작 기간 : 5월 28일 ~ 5월 30일<br>
Unity 2020.3.9 버전 이용<br>
목표 지점까지 빨리 도착해야 하는 캐주얼 레이싱 게임<br>
레이싱 게임으로 잘 알려진 <카트라이더>를 모티브로 함<br>
2021년 2학년 1학기 모바일 게임실무 수업에서 만든 게임 프로젝트<br>


<br><br>
<!-- GETTING STARTED -->
## 기능 :pencil2:

-  카운트 다운
-  현재 속도 출력
-  드리프트
-  부스터 
-  부스터 정렬
-  경로 이탈 시 시작 지점으로 이동
-  골인 - Best score, Score
-  체크포인트
-  esc 메뉴(게임 다시 시작, 메인 메뉴로 이동)
